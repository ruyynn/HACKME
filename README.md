<h2>📦 Instalasi</h2>

<details open>
<summary><b>🐧 Untuk Linux / Termux (Android)</b></summary>

```bash
# Update package
pkg update && pkg upgrade -y

# Install Python dan Git
pkg install python git -y

# Clone repository
git clone https://github.com/ruyynn/HACKME.git

# Masuk ke direktori
cd HACKME

# Jalankan tools
python hackme.py
```
</details><details> <summary><b>🪟 Untuk Windows</b></summary>
:: 1. Download Python dari python.org (centang "Add Python to PATH")
:: 2. Buka CMD atau PowerShell

git clone https://github.com/ruyynn/HACKME.git
cd HACKME
python hackme.py

</details><details> <summary><b>🍎 Untuk MacOS</b></summary>
# Install Python jika belum ada
brew install python

# Clone repository
git clone https://github.com/ruyynn/HACKME.git
cd HACKME

# Jalankan
python3 hackme.py

</details>
